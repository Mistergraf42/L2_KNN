La fonction supprimer permet de supprimer un point de l\textquotesingle{}arbre kd. Elle fait appel à la fonction profondeur\+\_\+noeud\+\_\+kd pour connaître la profondeur du point dans l\textquotesingle{}arbre kd. Elle parcourt l\textquotesingle{}arbre kd jusqu\textquotesingle{}à trouver le point à supprimer. Elle fait appel à la fonction comp\+Point pour comparer le point courant avec le point à supprimer. Si le point est trouvé elle le supprime et retourne l\textquotesingle{}arbre kd. Sinon elle affiche un message d\textquotesingle{}erreur.
